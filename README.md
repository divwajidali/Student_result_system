# 🎓 Student Result Management System

A beginner-friendly Python project that calculates a student's:

- ✅ Total Marks
- ✅ Percentage
- ✅ Grade
- ✅ Pass/Fail Status

This project is great for Python beginners to practice:
- Variables
- User Input
- Conditional Statements
- Arithmetic Operations
- f-strings

---

# 📌 Features

- Takes student name as input
- Accepts marks for 7 subjects
- Checks fail condition for every subject
- Calculates obtained marks
- Calculates percentage
- Assigns grade automatically
- Displays final status

---

# 📚 Subjects Included

- Urdu
- English
- Mathematics
- Islamiyat
- Physics
- Chemistry
- Biology

---

# 🏆 Grading System

| Percentage | Grade |
|------------|-------|
| 90% and above | A |
| 80% - 89% | B |
| 70% - 79% | C |
| 60% - 69% | D |
| 50% - 59% | E |
| Below 50% | F |
